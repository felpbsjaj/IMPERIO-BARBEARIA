# 👑 Império Barbearia — Gestão Pro

Sistema de gestão profissional para a Império Barbearia.

## Stack
- React 19 + TypeScript
- Tailwind CSS v4
- Firebase (Auth + Firestore)
- Recharts + Framer Motion

## Rodar local

```bash
npm install
npm run dev
```

Acessa em `http://localhost:3000`

## Deploy (Vercel)

```bash
npm run build
# ou conecta o repo no vercel.com — zero config
```

> Firebase já está configurado no código. Só subir e usar. 🚀
